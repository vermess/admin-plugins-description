Add custom descriptions to plugins in your admin panel on the plugins page and have more control over your installed plugins.

== Description ==

Admin Plugins Description plugin allows you to add your own description, a note about a specific plugin.

This way you don't have to remember what you used the plugin for or what functionality you use. 
You can make a note of the plugin modification information or any other important information about the selected plugin.

== Installation ==

Go to `Plugins -> Add New` section from your admin account and search for `Admin Plugins Description`.
